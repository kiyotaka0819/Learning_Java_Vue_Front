# Webポートフォリオ・システム (Java & Vue.js)
## 🥇 プロジェクト概要 (The System Vision)本リポジトリは、Java Spring Bootによる堅牢なバックエンドAPIと、Vue.js 3による**リアクティブなシングルページアプリケーション (SPA)**の構築を学ぶためのプロジェクトです。
- ポートフォリオサイトの土台として、自己紹介ページをトップに置き、そこから個別の練習用Webアプリケーションを配置するアプリケーション・ランチャーとしての役割を持たせています。
- 現在の到達点: Vue.jsとSpring Boot間の基本的なAPI連携（Hello World相当）および、DBの安定稼働環境構築。
- 目標: 技術選定の意図と、段階的なシステム構築のプロセスを記録すること。

## 🖥️ コア技術スタック (The Core Tech)カテゴリ技術使用目的フロントエンドVue.js3 (Composition API)SPA開発、リアクティブなデータ表示の習熟。
- ルーティングVue Routerサイト内でのスムーズな画面遷移と、個別アプリへのルーティング制御。
- バックエンドJava Spring Boot 3堅牢なRESTful APIの設計とビジネスロジックの実装。
- データベースPostgreSQLのクエリ出力とSQL操作の習熟。
- インフラAWS EC2 (Amazon Linux 2023)本番環境における Linux サーバー管理とデプロイ経験。

## ⚙️ 環境構築手順 (Getting Started)プロジェクトをローカル環境で起動するための手順を記載します。
1. プリセット (Prerequisites)Java SDK (21 LTS 推奨)Node.js & npm/yarnMariaDB サーバー（ローカルまたは開発用インスタンス）
2. バックエンド (Java) の起動プロジェクトをクローンします:Bashgit clone [GitHubリポジトリURL]
3. バックエンドディレクトリへ移動し、起動します:Bashcd backend-java-spring
./mvnw spring-boot:run
4. フロントエンド (Vue.js) の起動フロントエンドディレクトリへ移動します:Bashcd frontend-vue
依存関係をインストールし、開発サーバーを起動します:Bashnpm install
npm run dev
http://localhost:5173/ などでアクセス可能になります。

## 🧭 学習ロードマップとシステム拡張 (Phased Development)具体的な機能は後から追加しますが、システム全体の成長計画は以下の通りです。
- フェーズ主なテーマINTP-T的学習目標Phase 0 (骨格)環境構築、DB安定化安定稼働する基盤の確立と、最小限の連携APIの実装。
- Phase 1 (基盤)App #1 の実装と配置独立したアプリをポートフォリオ構造に組み込むルーティングとデータ連携の方法論確立。
- Phase 2 (セキュリティ)認証/認可システムシステム全体のセキュリティを担保する Spring Security (JWT) の実装と検証。
- Phase 3 (拡張性)アプリ追加と汎用化2つ目以降のアプリを効率的に追加するためのテンプレート化と設計の汎用化。
- Phase 4 (インフラ)Docker化とCI/CD自動化されたデプロイフローを構築し、システム管理の最適化を図る。
